# Space-Invaders-Reboot
A rendition of the classic Space Invader from 1978
<br><br>
Game Instructions:
<br><br><br>
-> Movement:
<br>
-> ** ARROW UP - Move up
<br>
-> ** ARROW DOWN - Move down
<br>
-> ** ARROW LEFT - Move left
<br>
-> ** ARROW RIGHT - Move right
<br><br><br>
-> Actions:
<br>
-> ** SPACE - Shoot
<br>
-> ** C - Hold to use shield
<br>
-> ** X - Special weapon 1
<br>
-> ** Z - Special weapon 2
<br>
